# CLAUDE.md

Guidance for working in this repository.

## Language

- Conversation with the user is normally in Japanese.
- Everything inside files (at least anything that will be committed) is in
  English — code, identifiers, comments, docstrings, commit messages, and any
  message strings printed by scripts. No Japanese in committed files.

## Comments

- Comment source code roughly block by block, so the intent of each block is
  clear on its own.
- Having more comments than typical source code is expected and fine.
- The goal: reading the source locally is enough to understand the purpose and
  intent of the surrounding code, without external context.
